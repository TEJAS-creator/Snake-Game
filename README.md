# Neon Snake Game 🐍

A complete, production-quality Snake Game built with HTML5, CSS3, and Vanilla JavaScript.

## 🌟 Features
- **Core Game Engine:** Built with HTML5 Canvas and `requestAnimationFrame` for a smooth, lag-free game loop.
- **Premium UI:** Dark theme aesthetics, neon/glow effects, glassmorphism containers, and glitch text animations.
- **Progressive Difficulty:** The game speeds up dynamically as your score increases (speeds up every 5 points).
- **Sound Effects:** Complete synthesized audio effects (eating, dying, leveling up) using the native Web Audio API—no external assets required.
- **Smart Controls:** Arrow keys for movement with instant buffering to prevent accidental self-collisions when pressing keys quickly. Spacebar to pause/resume.
- **Local Storage:** Automatically saves your high score across sessions.

## 🚀 How to Play
1. Clone this repository or download the files.
2. Open `index.html` in any modern web browser.
3. Click **START GAME** and use the arrow keys to navigate the snake.
4. Eat the red neon food to grow and increase your score.
5. Avoid hitting the walls or yourself!

## 🛠️ Built With
- **HTML5** (Structure & Canvas)
- **CSS3** (Styling, Keyframe Animations)
- **Vanilla JavaScript** (Game Logic, Physics, Web Audio API)
